🔒 Próximo Passo: Copiar Regras de Segurança

Após criar as collections, você vai precisar copiar as regras de segurança no Firebase Console.

rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /users/{userId} { allow read, write: if request.auth.uid == userId; } match /lojas/{lojaId} { allow read, write: if request.auth.uid == get(/databases/$(database)/documents/lojas/$(lojaId)).data.uid_owner; } match /produtos/{produtoId} { allow read, write: if request.auth != null; } match /pedidos/{pedidoId} { allow read: if request.auth.uid == resource.data.uid_owner; allow create: if request.auth != null; } match /invoices/{invoiceId} { allow read: if request.auth.uid == resource.data.uid_user; } } }

📖 Como Usar:

  1. Abra Firebase Console → Project Settings
  2. Copie sua config (todos os 6 campos)
  3. Cole aqui (ou clique em "Colar Config Completa")
  4. Clique em "Criar Collections"
  5. Aguarde confirmar ✓ todas
  6. Copie e aplique as Regras de Segurança
